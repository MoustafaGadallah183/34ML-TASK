


✅ Features
⚙️ MVVM Architecture — Clean separation between UI, business logic, and data services.

🧵 Swift Concurrency (async/await) — Seamless, modern asynchronous data loading using async/await.

📡 Remote API Integration — Fetches experiences from a backend service using URLSession.

📦 Local Offline Storage — Caches experience details locally using UserDefaults or custom in-memory storage to support offline access and avoid UI breakage without internet.

🖼️ Kingfisher — Efficient, async image loading and caching from remote URLs.

✨ SwiftUI Views — Fully built using SwiftUI for a reactive and modern UI.

🎥 Detail View — Rich experience detail screen with media overlays, dynamic counters, and graceful fallback UI.

📶 Offline Handling — Graceful UI degradation and placeholder content for when the user is offline.

📱 Adaptive Layout — Designed for all iPhone screen sizes with responsive layouts.
